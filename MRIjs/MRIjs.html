Help

MRIjs UI is a browser-based MRI physics simulator. Use the left panels to set the number of isochromats, the relaxation values, and the RF/gradient controls, then press Run TR to advance the simulation one repetition at a time. The left half of the screen shows the isochromats, and the right half shows time plots for phase, |Mxy|, and |Mz|.
